# Portfolio

My personal portfolio site, built with React, Vite, and deployed on Vercel.

## Features

- **Themeable** - All colors, fonts, and spacing as CSS variables to more easily swap themes.
- **Responsive** - Works across desktop, tablet, and mobile.
- **Project timeline** - A vertical timeline that lists my projects by date. Each card supports an image gallery, external links, and a skill list. Clicking an image opens a fullscreen lightbox with arrow navigation.
- **Experience cards** - Auto-sorted with my ongoing roles pinned to the top.
- **Hobby grid** - An Instagram-style square grid. Clicking a post opens a modal with a horizontal media carousel and a caption sidebar.
- **MonkeyType integration** - A slide-out tab on my hobbies page that displays my typing stats (30s personal best, total time typing, last test date) pulled from the MonkeyType API.
- **Serverless API proxy** - MonkeyType requests go through a Vercel serverless function for security.
-**Resume Page** - Contains resumes for SWE/UX Researcher roles. Download and viewing supported.
- **Contact page** — Mailto link for my email plus my GitHub and LinkedIn profiles.

## Tech Stack

React · Vite · React Router · React Icons · Vercel Serverless Functions
